## 🔬 MSc Thesis Highlight

**Evidence-Grounded Clinical Pharmacogenomics QA System**  
*Using LLMs + Hybrid Retrieval Augmentation*

- 🧠 Fine-tuned **LLaMA 3.1-8B** and **Qwen3-8B** using LoRA on 74,565 samples  
- ⚙️ Built a **6-stage Hybrid RAG pipeline** — lexical filtering → semantic re-ranking → evidence fusion  
- 📊 Achieved **91% accuracy & 0.87 macro F1** (vs 43.5% baseline — more than doubled)  
- 🗃️ Integrated two knowledge bases: CPIC (structured, via PySpark + DuckDB) & ClinPGx (unstructured text)  
- ✅ Evaluated using BERTScore, Cosine Similarity, LLM-as-a-Judge & human review
